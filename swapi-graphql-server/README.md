# Swapi Backend API

A simple API built from the swapi API to get star war data

### FEATURES

With the API you can:

- Retreive paginated list of people
- Search for people/person

### Technologies

- Node v12.0.0(Atleast)
- Graphql Apollo
- Express framework

### Installation

1. Clone the repository

2. Install dependencies
   `npm install`

3. Run app
   `npm start`

4. To build the app
   `npm run build`
